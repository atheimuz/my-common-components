# Collapse

Collapse은 확장/축소하여 더 많은 정보를 볼 수 있는 상위 수준 옵션 목록을 표시합니다.

## Import

```jsx
import { Collapse } from "my-common-components/react"
```

## 사용

```jsx live
<Collapse>
    <Collapse.Panel title="title1">content</Collapse.Panel>
    <Collapse.Panel title="title2">content</Collapse.Panel>
</Collapse>
```
