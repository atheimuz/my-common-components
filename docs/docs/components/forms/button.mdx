# Button

Button 구성 요소는 양식 제출, 대화 상자 열기, 작업 취소, 삭제 작업 수행 등의 작업이나 이벤트를 트리거하는 데 사용됩니다.

## Import

```jsx
import { Button } from "my-common-components/react"
```

## Usage

```jsx live
<Button theme="primary">Button</Button>
```

### 디자인

`theme` prop에 값을 전달하여 버튼의 디자인을 변경합니다.

`primary`, `secondary`, `tertiary`, `dashed`, `link`

```jsx live
<FlexRow>
    <Button theme="primary">primary</Button>
    <Button theme="secondary">secondary</Button>
    <Button theme="tertiary">tertiary</Button>
    <Button theme="dashed">dashed</Button>
    <Button theme="link">link</Button>
</FlexRow>
```

### 사이즈

`size` prop에 값을 전달하여 버튼의 크기를 변경합니다.

`small`, `middle`, `large`

```jsx live
<FlexRow>
    <Button size="small">small</Button>
    <Button size="middle">middle</Button>
    <Button size="large">large</Button>
</FlexRow>
```

### 비활성화

`disabled` prop에 값을 전달하여 비활성화 상태를 전환합니다.

```jsx live
<Button disabled>button</Button>
```

### 로딩

`isLoading` prop에 값을 전달하여 로딩 상태를 표시합니다.

```jsx live
<Button isLoading>button</Button>
```
