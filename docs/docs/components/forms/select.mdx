# Select

Select는 사용자가 미리 정의된 옵션에서 값을 선택할 수 있도록 하는 구성 요소입니다.

## Import

```jsx
import { Select } from "my-common-components/react"
```

## 사용

```jsx live
function Example() {
    const [value, setValue] = useState(1)
    return (
        <Select value={value} onChange={(val) => setValue(val)}>
            <Select.Option value={1}>옵션1</Select.Option>
            <Select.Option value={2}>옵션2</Select.Option>
            <Select.Option value={3}>옵션3</Select.Option>
        </Select>
    )
}
```

## 방향

`direction` prop에 값을 전달하여 위치를 변경합니다.

`horizontal`, `vertical`

#### horizontal

```jsx live
<RadioGroup name="radioA" value="red" direction="horizontal">
    <RadioGroup.Item value="red">빨강</RadioGroup.Item>
    <RadioGroup.Item value="yellow">노랑</RadioGroup.Item>
</RadioGroup>
```

#### vertical

```jsx live
<RadioGroup name="radioB" value="red" direction="vertical">
    <RadioGroup.Item value="red">빨강</RadioGroup.Item>
    <RadioGroup.Item value="yellow">노랑</RadioGroup.Item>
</RadioGroup>
```
